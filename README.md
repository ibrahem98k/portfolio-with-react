# Portfolio with React (Vite)

A modern, animated portfolio site built with React and Vite.  
It showcases:

- A hero section with stats and quick navigation
- Recent projects
- A “12 YEARS OF EXPERIENCE” timeline with video previews
- Premium tools section with real logos
- Design Thoughts articles, each with its own hero video
- Contact form section

All styling is done in a single [App.css](cci:7://file:///c:/Users/pc/new%20age/sawad-clone/src/App.css:0:0-0:0) for tight control over the layout and motion.

---

## Tech Stack

- **Framework:** React + Vite
- **Styling:** Plain CSS ([src/App.css](cci:7://file:///c:/Users/pc/new%20age/sawad-clone/src/App.css:0:0-0:0))
- **Animations / Scroll effects:** [AOS](https://michalsnik.github.io/aos/)
- **Icons:** Inline SVGs and external logo CDNs
